# 🚀 快速开始指南

## 一键启动

### macOS / Linux
```bash
./run.sh
```

### Windows
双击运行 `run.bat`

或在命令行中：
```cmd
python app.py
```

## 访问应用

启动后，在浏览器中访问：

```
http://localhost:5000
```

## 使用步骤

### 1️⃣ 获取 TikTok 视频链接

在 TikTok 应用或网页中：
- 点击视频的分享按钮
- 选择"复制链接"

### 2️⃣ 粘贴到应用中

在应用的输入框中粘贴链接，例如：
```
https://www.tiktok.com/@fishereffect/video/7563463737844174093
```

### 3️⃣ 获取评论

点击"获取评论"按钮，等待数据加载完成

### 4️⃣ 查看和导出

- **浏览评论**：滚动查看所有评论
- **搜索**：在搜索框输入关键词过滤评论
- **排序**：按点赞数或时间排序
- **导出**：点击导出按钮保存为 JSON 或 CSV

## 常见问题

**Q: 应用无法启动？**
A: 确保已安装 Python 3.8+ 和依赖包：
```bash
pip install -r requirements.txt
```

**Q: 无法获取评论？**
A: 检查：
1. 视频链接是否正确
2. 网络连接是否正常
3. 视频是否为公开状态

**Q: 如何停止应用？**
A: 在终端中按 `Ctrl + C`
